## Core Concept

A **Cybersocial Corporation (CSC)** is a new organizational form defined as:

```
CSC = ⟨N, R, P, G, M, E⟩
```

Where N=nodes, R=rules, P=participants, G=governance, M=mechanisms, E=economy

## 8 Axioms (A1–A8)

- A8: **Cognitive Augmentation** — AI integration is axiomatic to CSC

## Key Mechanisms

| Mechanism | Description |
|-----------|-------------|
| **PMIP** | Principle of Minimum Individual Participation — participation cost → 0 as SIC grows |
| **SES** | Socio-Economic Selection — evolutionary game-theoretic governance |
| **IPI** | Idea-Project-Implementation lifecycle (formal state machine) |
| **FRP** | Fork Resolution Protocol for governance conflicts |
