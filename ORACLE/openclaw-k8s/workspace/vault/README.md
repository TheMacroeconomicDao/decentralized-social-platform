# Gybernaty Knowledge Base

R&D knowledge vault for the GyberExperiment.

Maintained by Oracle (AI assistant) and Lll CIlician (founder).

## Structure

```
000-Overview/     — Maps of Content, project overview
100-Theory/       — CyberSocium theory, DAO design, academic work
200-Projects/     — Active project notes
300-Research/     — Research by domain (AI, blockchain, DeFi, swarm)
400-Daily/        — Daily research log (YYYY-MM-DD.md)
500-Archive/      — Completed work
```

## Conventions

- Frontmatter: `tags`, `created`, `status`
- Links: wikilinks `[[Note Name]]`
- Daily notes: `400-Daily/YYYY-MM-DD.md`
- Research results: append to relevant `300-Research/` subfolder
- Project updates: update `200-Projects/<name>/README.md`
