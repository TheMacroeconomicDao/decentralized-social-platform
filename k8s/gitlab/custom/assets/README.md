# Community Lab by Gybernaty - Ассеты

## Описание

Уникальные ассеты для Community Lab, созданные на основе дизайн-системы Gybernaty.

## Файлы

### `lab-logo.svg`
- **Размер**: 200x200px
- **Назначение**: Основной логотип лаборатории
- **Особенности**: 
  - Основан на оригинальном логотипе Gybernaty
  - Добавлены элементы лаборатории (колба, частицы)
  - Использует цветовую палитру Gybernaty (#d49d32, #42b8f3)
  - Glow эффекты для кибер-стилистики

### `lab-favicon.svg`
- **Размер**: 64x64px
- **Назначение**: Favicon для браузера
- **Особенности**: Упрощенная версия с колбой и буквой "L"

### `lab-icon-16.svg`
- **Размер**: 16x16px
- **Назначение**: Маленькая иконка для вкладок и UI элементов
- **Особенности**: Минималистичный дизайн

### `lab-header-logo.svg`
- **Размер**: 120x40px
- **Назначение**: Горизонтальный логотип для header GitLab
- **Особенности**: 
  - Иконка + текст "Community Lab by Gybernaty"
  - Оптимизирован для header навигации

## Цветовая палитра

- **Золотой (Gold)**: `#d49d32` → `#f4c430` → `#ff6b35`
- **Синий (Blue)**: `#42b8f3` → `#00d4ff`
- **Темный фон**: `#001019`, `#072b40`
- **Белый текст**: `#e1e1e1`

## Использование в GitLab

1. **Логотип**: Загрузить `lab-logo.svg` в Admin Area → Appearance → Logo
2. **Favicon**: Загрузить `lab-favicon.svg` в Admin Area → Appearance → Favicon
3. **Header**: Использовать `lab-header-logo.svg` для кастомного header
4. **Иконки**: Использовать `lab-icon-16.svg` для UI элементов

## Интеграция

Ассеты автоматически применяются через скрипт `apply-gybernaty-branding.sh` или вручную через веб-интерфейс GitLab.
